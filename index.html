API- Weather Service

Project: Weather Service API

Developed a RESTful API service that provides real-time weather information for any location worldwide. The API integrates with third-party weather data providers to fetch and deliver data such as temperature, humidity, wind speed, and weather conditions.

The service supports multiple endpoints to handle various requests, such as current weather data, historical data, and forecast data. It includes robust error handling, caching for performance optimization, and user authentication for secure access.

Technologies: Python, Flask, Redis, PostgreSQL, REST API, Docker, AWS

the Maze Game Project

Developed a dynamic maze game that challenges players to navigate through randomly generated mazes of varying complexity. The game includes an algorithm that dynamically creates unique mazes each time, providing a different experience for every playthrough.

The backend logic handles maze generation using algorithms like Depth-First Search (DFS) and Prim's algorithm, ensuring that each maze has a single, solvable path. The game also includes a timer and scoring system, which keeps track of player progress and ranks their performance.

Technologies: Python, Flask, JavaScript, HTML5 Canvas, WebSockets
